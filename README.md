# flour

Moving square block in terminal thing.

## Building

### Prerequisites

- GNU Make
- GCC or any C17 compiler that works
- ncursesw (>= 6.3)

### Compiling

    $ git clone https://github.com/iarkn/flour.git && cd flour
    $ make

The build output is in the `build` directory. To run the program, execute
`./build/flour`.
